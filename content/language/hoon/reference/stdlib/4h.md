# 4h: Parsing (ASCII Glyphs) {#4h-parsing-ascii-glyphs}

## `++ace` {#ace}

Parse space

Parses ASCII character 32, space.

#### Source {#source}

```hoon
++  ace  (just ' ')
```

#### Examples {#examples}

```
> (scan " " ace)
' '
```
